# Trainable forecast data with weather

Generated by `scripts/prepare_trainable_forecast_weather.py`.

Each row uses information available at date `ds` to predict a future target date.

Targets:

- `target_h{h}`: future weighted average price after h days.
- `target_return_h{h}`: future percentage change from current `y`.
- `target_up_h{h}`: classification target for price increase.
- `target_rise_ge_5pct_h{h}`: classification target for >=5% increase.
- `target_rise_ge_10pct_h{h}`: classification target for >=10% increase.

Split rule: train/test is determined by `target_date`, so rows whose labels fall in the original test period are held out.
